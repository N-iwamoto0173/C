# 名簿管理アプリ　仕様書
## 【概要】
### 目的
* 企業実習の課題として作成
* C言語の基礎を習得
* ポインタ、メモリ管理、リスト構造の理解
### 開発環境
* 使用ツール：サクラエディタ
* 言語：C言語（コンパイラの都合上.cppとして作成）
* コンパイラ：bcc32（Borland C++）/gcc（Linuxでの移植確認用）
* 文字コード：Shift-JIS (bcc32環境) / UTF-8 (Linux環境)
## 【機能一覧】
1. 登録：名簿情報の入力、社員コード昇順で登録
    * 入力された社員コードが既に登録されている場合、メインメニューに戻る（一覧表示で確認したい）
2. 変更：名簿に登録されている各項目について変更
3. 削除：登録されている名簿を削除
    * 削除前に確認メッセージ表示
4. 一覧表示：登録されている名簿の一覧表示
5. 保存：登録されている名簿情報をCSV形式で保存
6. 読み込み：CSV形式で保存した名簿情報をアプリ内へ持ってくる
    * 社員コードが重複しているデータは読み込みをスキップ
## 【データ構造】
* 構造体（struct meibot）を単方向連結リストで管理

| 項目 | 型 | 説明 |
| :--- | :--- | :--- |
| 社員コード | char[6] | 5桁、一意、0埋め |
| 氏名 | char[31] | 30文字まで入力可 |
| 性別・年齢 | unsigned char | 性別は0または1、年齢は0～127歳を2進数変換、合わせて8ビットで管理 |
| 電話番号 | char[16] | 15文字まで入力可 |

## 【画面遷移と操作方法】
* 1つ前の演習で作成した`文字列入力用サブルーチン`を使用
    * 英数字（と空白）入力/数字（と空白）入力/なんでも入力の3モードを実装
    * Enterキー、Backspaceキーに対応
    * この演習で0/1、Y/Nの入力モードを追加
* メインメニューで各機能に対応した数字を入力すると、該当の処理をする
* メインメニューで９を入力するとアプリが終了する

---
### 【追加実装機能について】
* Windowsだけでなく、Linuxでも動く移植性の高いコードへ修正
* 文字列入力の際、左右のカーソル移動を実装
* 文字列入力の際、Insertキーに対応
* ファイル読み込み時に不正なデータは読み込まずスキップする
* ファイル読み込みを一覧から選択してできるように修正
    * ファイルへのデータ保存時に拡張子.txtをつけるよう誘導
* 一覧表示メニューに名簿のソート機能を実装
* 既存名簿の検索機能を実装
    * あいまい検索（部分一致、大文字小文字の区別なし）
* 文字列入力の際画面表示がおかしくなることに対する対応
    * 各機能の処理が終了し、メインメニューに戻る際に画面リセット
