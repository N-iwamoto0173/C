# 名簿管理アプリ

職業訓練校のカリキュラムの一環で、企業実習に行った際に作成したもの。

サクラエディタ、C言語を使用。(保存形式として.cppを指定されたのでファイルはC++になっている)

課題のゴールは「リスト構造の理解」

実装機能は以下の通り。
* (昇順)登録
* 項目を指定して変更
* 一件または全件削除
* 一覧表示（ソート機能あり）
* 名簿をCSV形式で保存
* .txtファイルを読み取り可能なファイルとし、番号で選択して読み込み
